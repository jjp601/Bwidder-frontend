# Bwidder Project - Frontend
## Summary
The Bwidder application is a full-stack React application that consumes a RESTFUL API built with Express, Nodejs, and MongoDB. The goal of the project was to build a a complete full-stack React and Redux Application that uses a custom built API. The project utilizes MongoDB for a Non-relational database and Mongoose.js as an Object Data Modeler. The fronted is built with React and uses Redux for state managment as well as React Router for routing. The application uses bcrypt and JSON Web Tokens for Authentication and Authorization. The Bwidder application is a social app for people looking to buy and sell items over a social network. You may visit the deployed application [here](https://bwidder-frontend.herokuapp.com/).
